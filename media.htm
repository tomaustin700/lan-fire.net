Building a HTPC:
Many of you will be familiar with a new bread of PC promoted by intell and Microsoft called the HTPC or home theater PC. These computers have been around for 4 years or more but have only recently become popular with the launch of windows xp media center addition. This is windows xp in a new shell which is easy to read on a TV screen from a distance and navigate with a remote control. With windows media center edition manufactures have had an incentive to start to pre package Pc's with HTPC functionality. The main problem to this is that media center is not available to by except with a new computer from a major supplier, an expensive proposition at best, thus the purpose of this guide is to educate you in how to build a HTTPC


What constitutes a HTPC?
A HTPC is a computer capable of capturing live video and audio streams and storing them or redistributing them on a medium such as DVD-RW. Many HTPC's have electronic program guides similar to the ones with freeveiw and sky making them subtable for replacing the TV times lost behind the sofa.


What hardware does a HTPC contain?
 Normal Pc components (ram,mobo,.ECT)
Some kind of motion picture capture card/ tuner
A remote control device Eg. TV wonder (don't want to be getting up all the time do you?)
Large capacity hard disk(s) ( video files are huge!)
Relevant software


As you can see HTPC's are normal computers with a few add aditional components so an existing PC could be converted or one could be built from scratch. Building from scratch is the more common option as many people object to a big beige tower next to there TV! Because of this SFF (small form factor) computers have become widespread. A small form factor computer is anything bellow standard ATX size (micro ATX, shuttles .ext)
